# Sopia Landing Page

Marketing landing page for Sopia - AI-powered omnichannel CRM platform for technical support teams.

## Features

- Modern, responsive design
- Brand-consistent styling using Sopia color palette
- Sections: Hero, Features, How It Works, Benefits, Trust Indicators, Pricing, CTA, Footer
- Built with React, TypeScript, Vite, and Tailwind CSS

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Design System

Colors match the support-platform app:
- Primary: Russian Violet (#462255)
- Secondary: Delft Blue (#313b72)
- Success: Mint (#62a87c)
- Accent: Light Green (#7ee081)

Typography: Inter font family
