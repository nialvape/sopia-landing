# Sopia: CRM Omnicanal con IA para Soporte Técnico Moderno

Sopia es la plataforma definitiva diseñada para transformar la atención al cliente de equipos técnicos. Unificamos tus canales de comunicación y potenciamos tu productividad con Inteligencia Artificial.

## ¿Qué hace Sopia por tu negocio?

- **Unificación de Canales**: Gestioná WhatsApp e Instagram desde un solo panel. Próximamente Email y Web Chat.
- **Automatización con IA**: Agentes inteligentes que responden 24/7 y resuelven dudas frecuentes de forma instantánea.
- **Gestión Inteligente de Tickets**: Asignación automática de casos según el departamento y carga de trabajo del equipo.
- **Acción Inmediata (Next-Best-Action)**: La IA no solo responde, sino que ejecuta procesos automáticamente: crea órdenes, genera facturas y notifica estados.
- **Analytics y Métricas**: Tablero de control completo para monitorear tiempos de respuesta y satisfacción del cliente en tiempo real.

## ¿Por qué elegir Sopia?

1. **Eficiencia**: Reducí el tiempo de resolución de problemas mediante automatizaciones que van de la charla a la acción.
2. **Escalabilidad**: Crecé sin necesidad de aumentar linealmente tu equipo de soporte gracias al apoyo de agentes de IA.
3. **Experiencia del Cliente**: Brindá respuestas consistentes y profesionales en todos tus canales oficiales (WhatsApp Business API e Instagram).

### Detalles del Proyecto (Desarrollo)

Para configurar el entorno de desarrollo localmente:

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```
